DRAGONS Tools¶

Python tools for dealing with Meraxes semi-analytic model output and associated post-processing.

  • Free software: GNU General Public License v3 (GPLv3)

  • Documentation: http://www.ph.unimelb.edu.au/~smutch/docs/dragons .

  • Lead author: Simon Mutch (The University of Melbourne)

Features¶

  • Functions for reading and parsing Meraxes output (meraxes)

  • Functions for reading and parsing gbpCode nbody output (nbody)

  • Common processing tasks for analysing simulation/model data (munge)

  • Tools for common plotting tasks (plotutils)

Tests¶

Whilst all of the code in this package has been heavily tested in the field by a number of people, it is sorely lacking unit tests. This is a high priority for this package. Anyone who is interested or willing to help is encouraged to contact the authors.
